---

## ¿Qué problema resuelve Git?

Git resuelve el problema de controlar versiones de un proyecto.

Sin Git, una persona puede terminar guardando archivos como:

- proyecto-final;
- proyecto-final-2;
- proyecto-ahora-si;
- proyecto-no-tocar.

Con Git, cada avance importante se guarda mediante commits. Eso permite saber qué cambió, cuándo cambió y por qué cambió.

---

## ¿Qué diferencia hay entre Git y GitHub?

Git es la herramienta local de control de versiones.

GitHub es una plataforma online donde se alojan repositorios Git. Permite subir el proyecto a la nube, trabajar con ramas, issues, pull requests y mostrar evidencia profesional.

En este repositorio usé Git para trabajar localmente y GitHub para publicar, documentar y gestionar el avance de la materia.
