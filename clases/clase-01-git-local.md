Clase 1 - Git, GitHub y Terminal Profesional

Git es una herramienta local de control de versiones que registra la historia de cambios de un proyecto.

GitHub es una plataforma online donde se alojan repositorios Git, permitiendo respaldo remoto, colaboración y exposición profesional del código.

Un repositorio es una carpeta de proyecto controlada por Git.

El working directory es la zona donde trabajo y modifico archivos.

El staging area es la zona donde preparo los cambios que quiero guardar.

El repository es el historial definitivo donde quedan registrados los commits.

Un commit es un punto de control profesional dentro del proyecto.
