# Clase 2 - GitHub, repositorio remoto y README profesional

## Objetivo de la clase

Conectar un repositorio local con GitHub y publicar la primera evidencia académica del programa.

## Conceptos trabajados

- Repositorio local.
- Repositorio remoto.
- GitHub.
- Rama `main`.
- Remoto `origin`.
- Comando `git push`.
- Archivo `README.md`.

## Comandos utilizados

```bash
git branch -M main
git remote add origin URL_DEL_REPOSITORIO
git remote -v
git push -u origin main
touch README.md
git add README.md
git commit -m "Agregar README profesional"
git push
