# Clase 3 - Git diff y flujo profesional de cambios

## Objetivo de la clase

Aprender a revisar modificaciones antes de guardarlas en el historial del repositorio.

## Conceptos trabajados

- Modificación de archivos existentes.
- Estado del repositorio.
- Cambios no preparados.
- Cambios preparados.
- Diferencia entre `git diff` y `git diff --staged`.

## Comandos utilizados

```bash
git status
git diff
git add README.md
