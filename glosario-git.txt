Glosario básico de Git

Git:
Herramienta local de control de versiones que registra la historia de cambios de un proyecto.

GitHub:
Plataforma online donde se alojan repositorios Git para respaldo remoto, colaboración y exposición profesional.

Repositorio:
Carpeta de proyecto controlada por Git.

Working Directory:
Zona de trabajo donde se crean, editan o eliminan archivos.

Staging Area:
Zona intermedia donde se preparan los cambios antes de guardarlos definitivamente.

Commit:
Punto de control dentro del historial del proyecto.

Branch:
Línea de trabajo independiente dentro de un repositorio.

HEAD:
Referencia que indica en qué commit o rama estamos ubicados actualmente.

Remote:
Repositorio remoto alojado en una plataforma online como GitHub.

Origin:
Nombre convencional que Git utiliza para identificar el repositorio remoto principal.

Push:
Acción de subir commits desde el repositorio local hacia el repositorio remoto.

Diff:
Comparación entre versiones de un archivo o conjunto de cambios. Permite revisar qué se modificó antes de preparar o guardar los cambios.
