## Programa Complementario de Formación Frontend - UTN

Repositorio académico-profesional correspondiente a la **Materia 1: Git, GitHub, Terminal e IA Profesional**, dentro del Programa Complementario de Formación Frontend diseñado para complementar la Tecnicatura Universitaria en Programación de la UTN.

> Objetivo: construir una base sólida de trabajo profesional con Git, GitHub, terminal, documentación, trazabilidad, Issues, Pull Requests, bitácora técnica y uso responsable de inteligencia artificial.


## Metodología oficial del programa

Cada materia del Programa Complementario de Formación Frontend tendrá su propio repositorio actualizado en GitHub.

Cada repositorio deberá incluir:

- clases documentadas;
- recursos de consulta;
- reflexiones personales;
- bitácora técnica;
- commits claros;
- uso de ramas;
- issues;
- pull requests;
- documentación profesional;
- trabajo integrador final.

El objetivo no es solamente guardar archivos, sino construir una evidencia ordenada, progresiva y defendible del aprendizaje realizado.

---

## Contenidos trabajados

Durante esta materia se trabajaron los siguientes temas:

- Diferencia entre Git y GitHub.
- Repositorio local y repositorio remoto.
- Working directory, staging area y repository.
- Commits e historial de versiones.
- Uso de `git status`, `git add`, `git commit` y `git log`.
- Revisión de cambios con `git diff` y `git diff --staged`.
- Ramas de trabajo.
- Reorganización de archivos con `git mv`.
- Merge y fast-forward.
- Sincronización con GitHub.
- Issues como tareas de trabajo.
- Pull Requests como mecanismo de revisión.
- Uso de `Closes #n` para cerrar Issues automáticamente.
- Política personal de uso profesional y ético de IA.
- Bitácora técnica de la materia.
