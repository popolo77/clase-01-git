# Clase 01 - Git, GitHub y Terminal Profesional

Repositorio académico correspondiente a la Materia 1 del Programa Complementario de Formación Frontend, diseñado para complementar la Tecnicatura Universitaria en Programación de la UTN.

## Objetivo del repositorio

El objetivo de este repositorio es registrar la primera práctica formal de control de versiones utilizando Git y GitHub.

En esta clase se trabajaron los conceptos fundamentales de Git, incluyendo:

- repositorio local;
- working directory;
- staging area;
- commits;
- historial de versiones;
- repositorio remoto en GitHub.

## Contenidos trabajados

- Diferencia entre Git y GitHub.
- Inicialización de un repositorio local con `git init`.
- Creación de archivos desde terminal.
- Seguimiento de archivos con `git status`.
- Preparación de cambios con `git add`.
- Registro de cambios con `git commit`.
- Visualización del historial con `git log --oneline`.
- Conexión con repositorio remoto.
- Subida del proyecto a GitHub con `git push`.

## Archivos incluidos

- `resumen.txt`: resumen conceptual de la clase.
- `objetivos.txt`: objetivos personales de formación Frontend.
- `glosario-git.txt`: definiciones básicas de Git.
- `reflexion-clase-01.txt`: reflexión sobre la importancia de Git en la formación profesional.
